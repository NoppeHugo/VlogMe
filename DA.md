# 🎨 Direction Artistique — Inspirée de SuperShrimp.io
Pour une app iOS de vlogging

## 1. COULEURS
| Élément | Valeur |
|---|---|
| Couleur primaire (accent) | `#FF6B4A` (orange-corail chaleureux, vivant) |
| Fond principal | `#FFFFFF` (blanc pur) ou très légèrement crème |
| Fond secondaire | `#F9F6F2` (blanc cassé/beige doux) |
| Texte principal | `#1A1A1A` (presque noir) |
| Texte secondaire | `#6B7280` (gris neutre) |
| Bordures / séparateurs | `#E5E7EB` (gris très clair) |
| Badges / tags | fond coloré léger avec texte assorti |

## 2. TYPOGRAPHIE

**Titres (H1)**
- Serif moderne ou sans-serif bold, très grande taille, impactant — style editorial/magazine
- Font-weight: 800–900 (ExtraBold / Black)
- Taille: 56–72px desktop, 36–48px mobile

**Sous-titres (H2)**
- Sans-serif semi-bold
- Font-weight: 600–700
- Taille: 28–36px

**Corps de texte**
- Sans-serif regular, lisible et aéré
- Font-weight: 400–500
- Taille: 16–18px, line-height: 1.6–1.7

**CTA / Boutons**
- Sans-serif bold, majuscules ou capitalisé

**Stack typographique recommandée :** Inter ou Geist (sans-serif) + possiblement un serif display pour les grands titres.

## 3. LAYOUT & STRUCTURE

- Full-width sections qui s'alternent : fond blanc / fond beige clair
- Centré, aéré — beaucoup de whitespace. Jamais surchargé
- Largeur de contenu max : ~720–900px, centré horizontalement
- Sections bien délimitées avec une hiérarchie claire : Hero → Features → Pricing → FAQ → Footer
- Hero minimaliste : grande accroche, sous-titre, CTA principal, une image de produit directement en dessous

## 4. COMPOSANTS UI

### Boutons
**Bouton primaire**
- Fond `#FF6B4A`, texte blanc, border-radius: 8–12px
- Padding: 14px 28px, font-weight: 700
- Hover: légèrement plus sombre

**Bouton secondaire**
- Bordure fine, fond transparent, texte foncé

### Cards / Blocs features
- Fond blanc ou légèrement grisé
- Border-radius: 16–20px
- Ombre subtile (box-shadow légère)
- Icône ou GIF animé en haut de la card
- Titre bold + bullet points avec checkmark (✓) ou icône simple

### Tags / Badges
- Petits, arrondis (pill shape)
- Fond coloré pastel + texte sombre
- Utilisés pour des statuts ("LIVE", "New", etc.)

### Section pricing
- Une seule card centrale, mise en valeur
- Prix barré (ancien prix) + prix actuel plus grand et gras
- Liste des features avec checkmarks
- CTA bouton en dessous
- Mention urgence/scarcité en petit texte sous le bouton

## 5. ICONOGRAPHIE & VISUELS

- Emoji utilisés de façon assumée et fun dans le texte (🎥 📱 ✨) — mais avec parcimonie
- Screenshots/GIFs animés du produit directement dans les sections features
- Images lifestyle illustratives (personne utilisant l'app dans un contexte réel)
- Pas d'illustrations vectorielles complexes — privilégier le réel et le direct

## 6. TONE OF VOICE (impacte le copywriting & micro-copy)

- Direct et sans bullshit : "Fix your posture" pas "Optimize your ergonomic experience"
- Humain et proche : comme si c'était un ami qui te parle
- Un peu fun avec des touches d'humour (emoji, tournures légères)
- Honnête : chiffres concrets, transparence sur le prix, garantie claire
- Urgence douce : "Launch price — will increase soon" sans être agressif

## 7. ANIMATIONS & MICRO-INTERACTIONS

- Scroll animations subtiles : fade-in + légère montée (translateY)
- Texte animé dans le Hero : alternance de mots en rotation (typewriter / crossfade)
- GIFs en loop pour démontrer les features (pas de vidéos lourdes)
- Transitions douces : 200–300ms, easing naturel
- Pas d'animations tape-à-l'oeil — tout reste sobre et fonctionnel

## 8. ESPACEMENT (Spacing System)

| Élément | Valeur |
|---|---|
| Base unit | 4px |
| Sections | padding vertical 80–120px |
| Cards gap | 24–32px |
| Éléments | gap 12–16px |
| Texte H1 → sous-titre | margin-bottom 16–24px |

## 9. AMBIANCE GÉNÉRALE (pour l'app iOS)

Pour l'app de vlogging, adapter ainsi la DA :

| Élément | Adaptation iOS |
|---|---|
| Fond | Dark mode élégant recommandé (`#0D0D0D`) pour l'expérience caméra |
| Accent | Conserver `#FF6B4A` pour les CTA et éléments actifs |
| UI de caméra | Interface minimaliste, contrôles discrets sur les bords |
| Typo | SF Pro (natif iOS) pour l'app, cohérent avec l'approche clean |
| Bouton record | Rond, grand, centré, pulsation légère à l'activation |
| Cards/Miniatures | Coins arrondis 12–16px, ombres douces |
